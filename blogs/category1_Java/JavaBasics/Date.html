    # 第一代日期类

    1. Date 精确到毫秒，代表特定的瞬间
    2. SimpleDateformal 格式和解析日期的类
      格式化和解析日期的具体类，允许进行格式化（日期->文本）、解析（文本->日期）和规范化。
    //1.获取当前系统时间java.util.Date
//2.默认输出的其他日期格式，通常需要对格式进行转换
Date date = new Date();//获取当前系统时间
Date date1 = new Date(987656);//通过指定毫秒数得到时间
System.out.println(date);
System.out.println(date1);
//创建SimpleDateFormat对象，可以指定相应的格式
SimpleDateFormat sdf = new SimpleDateFormat("hh:mm:ss E");
String fotmat = sdf.format(date);
System.out.println(fotmat);
//可以把一个格式的String 转换成相应的Date
//得到Date 仍然在输出时，还是按照国外的形式，如果希望指定格式输出，需要转化
//在吧String-> Date,使用sdf的格式和给的String的格式一样,否则会抛出一个转换异常
String s = "2000年01月01日 10:20:30 星期一";
Date parse = sdf.parse(s);
System.out.println(sdf.format(parse));

    2. Calender类是一个抽象类，它为特定瞬间与一组诸如YEAR、MONTH、DAY_OF_MONTH、HOUR等日历字段之间的转换提供了一些方法， 并为操作日历字段（例如获得下星期的日期）提供了一些方法。

    //1.Calendar是一个抽象类，并且构造器是private
//2.可以通过getInstance（）来获取实例
//3.提供大量的方法和字段
Calendar c = Calendar.getInstance();//创建日历对象
System.out.println(c);
//获取日历对象的某个日历字段
System.out.println("年："+c.get(Calendar.YEAR));
System.out.println("月："+c.get(Calendar.MONTH)+1);//Calendar 返回月时，从0开始编号
System.out.println("日："+c.get(Calendar.DAY_OF_MONTH));
System.out.println("小时："+c.get(Calendar.HOUR));
System.out.println("分钟："+c.get(Calendar.MINUTE));
System.out.println("秒："+c.get(Calendar.SECOND));
//Calendar没有专门的格式化方法，需要自己组合
System.out.println(c.get(Calendar.SECOND)+"-"+c.get(Calendar.MINUTE));

    # 第三代日期类

    Calendar存在的问题：可变性，偏移性（年份从1900开始，月份从0开始），线程不安全；不能处理闰秒（每隔两天多出1秒）

    1. LocalDate（日期/年月日）、LocalTime（时间/时分秒）、LocalDateTime（日期时间/年月日时分秒）JDK8加入
    3. DateTimeFormatter格式日期类 类似于SimpleDateFormat
      DateTimeFormat  dtf  =  DateTimeFormatter.ofPattern(格式); String  str  =  dtf.format（日期对象）;
    public class LocalDate_ {
    public static void main(String[] args) {
        //第三代日期
        //1.使用 now（）返回表示当前日期时间的对象
        LocalDateTime ldt = LocalDateTime.now();//LocalData.now()//LocalTime()
        System.out.println(ldt);
        System.out.println("年="+ldt.getYear());
        System.out.println("月="+ldt.getMonth());
        System.out.println("月="+ldt.getMonthValue());
        System.out.println("日="+ldt.getDayOfMonth());
        System.out.println("时="+ldt.getHour());
        System.out.println("分="+ldt.getMinute());
        System.out.println("秒="+ldt.getSecond());

        LocalDate now = LocalDate.now();//可以获取年月日
        LocalDate now2 = LocalDate.now();//可以获取时分秒

        //2.使用DateTimtFormatter 对象进行格式化
        //创建DateTimeFormatter对象
        DateTimeFormatter dateTimeFormatter = DateTimeFormatter.ofPattern("yyyy年MM月dd日 HH小时mm分钟ss秒");
        String format = dateTimeFormatter.format(ldt);
        System.out.println("格式化的日期"+format);
    }
}

    public class Instant_ {
    public static void main(String[] args) {
        //1.通过静态方法now（）获取表示当前时间戳的对象
        Instant now = Instant.now();
        System.out.println(now);
        //2.通过from 可以把Instant转成Date
        Date date = Date.from(now);
        System.out.println(date);
        //3.通过date的toInstant（）可以把date转成Instant对象
        Instant instant = date.toInstant();
        System.out.println(instant);
        ////第三代更多方法
        //提供了plus 和minus方法可以对当前的时间进行加或减
        LocalDateTime localDateTime = ldt.plusDays(890);
        System.out.println(dateTimeFormatter.format(localDateTime));//格式化
        //看看在345分钟前
        LocalDateTime localDateTime1 = ldt.minusMinutes(3456);
        System.out.println(dateTimeFormatter.format(localDateTime1));//格式化
        //等等
    }
}
