    # this

    this ：哪个对象调用，this就代表哪个对象。

    1. 访问成员方法的语法：this.方法名（参数列表）

    2. 访问构造器语法：只能在构造器中使用（即只能在构造器中访问另一个构造器），不可用在普通方法

    3. this关键字可以用来访问本类的属性

    public class This01 {
    public static void main(String[] args){
        Dog dog1 = new Dog(3,"一");
        dog1.Print();
    }
}
class Dog{
    int age;
    String name;
    //this ；哪个对象调用，this就代表哪个对象
    public Dog(int age ,String name/*int Aage,String Nname*/){
        //name = Nname;
        //age = Aage;
        this.age = age;//this.age 当前对象的属性
        this.name = name;
    }
    public void Print(){
        System.out.println(name+"\t"+age+"\t");
    }
}

class X{
    int age;
    String num;
    //1.访问成员方法的语法：this.方法名（参数列表）
    public void f1(){
        System.out.println("f1()方法");
    }
    public void f2(){
        System.out.println("f2()方法");
        //调用本类的f1
        //第一种
        f1();
        //第二种
        this.f1();
    }
    //2.访问构造器语法：只能在构造器中使用（即只能在构造器中访问另一个构造器），不可用在普通方法
    public X(){
        //访问X（int n,String m）
        this(100,"一");//this（）在第一行，对this（）调用必须是构造器中的第一个语句
        System.out.println("X()构造器");
    }
    public X(int n,String m){
        System.out.println("X(int n,String m)构造器");
    }
    //3.this关键字可以用来访问本类的属性
    public void f3(){
        //传统方式
        System.out.println(age+num);
        //this
        System.out.println(this.age+this.num);
    }
}

    # super

    super主要用来访问：

    1. 访问父类的属性（成员变量），但不能访问父类的private属性
    2. 访问父类的方法，不能访问父类的private方法
    3. 访问父类的构造器：super（参数列表），只能放在第一句，只能出现一句
      • 访问无参构造器：super（）
      • 有参构造器：super（形参列表）

    用途：

    1. 分工明确，父类属性由父类初始化，子类属性由子类初始化
    2. 子类中有和父类方法重名的成员（属性或方法），为了访问父类成员，必须通过super。

    super的访问不限于直接父类。如果父类的父类和本类有同名成员，也可以通过super进行访问；如果有父类和父类的父类有同名成员，则遵循就近原则。

    package test.Super_;

public class SuperTest {
    public static void main(String[] args) {
        B b = new B();
        b.hello1();
        b.hello2();
        b.test();//子类的method方法
    }
}

/*
1. 访问父类的属性（成员变量），但不能访问父类的private属性
2. 访问父类的方法，不能访问父类的private方法
3. 访问父类的构造器：super（参数列表），只能放在第一句，只能出现一句
   - 访问无参构造器：super（）
   - 有参构造器：super（形参列表）

用途：
1. 子类中有和父类方法重名的成员（属性或方法），为了访问父类成员，必须通过super。
2. 分工明确，父类属性由父类初始化，子类属性由子类初始化

super的访问不限于直接父类。如果父类的父类和本类有同名成员，也可以通过super进行访问；如果有父类和父类的父类有同名成员，则遵循就近原则。
 */
class A {
    private int n1 = 1;
    protected int n2 = 2;
    int n3 = 3;
    public int n4 = 4;

    public A() {
        System.out.println("父类的无参构造方法被调用");
    }

//    public A(int n1, int n2, int n3, int n4) {
//        this.n1 = n1;
//        this.n2 = n2;
//        this.n3 = n3;
//        this.n4 = n4;
//        System.out.println("父类的有参构造方法被调用");
//    }

    private void m1() {
        System.out.println("private类型的m1方法被调用");
    }

    protected void m2() {
        System.out.println("protected类型的m2方法被调用");
    }

    void m3() {
        System.out.println("默认类型的m1方法被调用");
    }

    public void m4() {
        System.out.println("public类型的m1方法被调用");
    }

    public void method() {
        System.out.println("父类的method方法");
    }
}

class B extends A {
    public String name = "lpy";
    //3. 访问父类的构造器：super（参数列表），只能放在第一句，只能出现一句
    //   - 访问无参构造器：super（）
    //   - 有参构造器：super（形参列表）
    public B() {
        super();//调用父类的无参构造（默认）
        System.out.println("子类的无参构造方法被调用");
//        super(11,12,76,44);//调用父类的有参构造
//        System.out.println("子类的有参构造方法被调用");
    }

    //1. 访问父类的属性（成员变量），但不能访问父类的private属性
    public void hello1() {
        System.out.println(/*super.n1*/+super.n2 + super.n3 + super.n4);//2+3+4=9
    }

    //2. 访问父类的方法，不能访问父类的private方法
    public void hello2() {
//        super.m1();//错误
        super.m2();
        super.m3();
        super.m4();
    }

    @Override
    public void method() {
        System.out.println("子类的method方法");
    }
    //用途1.子类中有和父类方法重名的成员（属性或方法），为了访问父类成员，必须通过super。
    public void test() {
        super.method();//访问父类同名的方法
        this.method();//访问子类（本类）同名的方法
    }
}
//运行结果：
//父类的无参构造方法被调用
//子类的无参构造方法被调用
//9
//protected类型的m2方法被调用
//默认类型的m1方法被调用
//public类型的m1方法被调用
//父类的method方法
//子类的method方法

    public class A {
    public int n1 = 100;
    public void cal(){
        System.out.println("A类的cal方法");
    }
}
public class B extends A{
    public int n1 = 888;
    public void cal(){
        System.out.println("B类的cal方法");
    }
    public void sum(){
        System.out.println("B类的sum方法");
        //调用父类A的cal方法
        //子类没有cal方法，可以用三种方式

        //找cal方法（1、cal();2、this.cal()），顺序是：本类->父类->父类的父类，直到object类
        //查找过程中，找到了，但不能访问（private），则报错
        //未找到，提示方法不存在
        //cal();
        //this.cal();//等价与cal（）

        //3、super.cal（）方法
        super.cal();//直接查找父类，其他规则一样

        //演示访问属性的规则
        //找n1属性，顺序是：本类->父类->父类的父类，直到object类（与方法相同）
        System.out.println(n1);
        System.out.println(this.n1);
        System.out.println(super.n1);//直接查找父类，其他规则一样

        //super访问不限于直接父类，如果爷类与本类中有同名的成员，也可以使用super去访问爷爷类的成员，如果多个上级类中都有同名成员，使用super访问遵循就近原则
    }
}
public class Super02 {
    public static void main(String[] args) {
        B b = new B();
        b.sum();
    }
}
